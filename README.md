# 🌐 Portfolio — Farooque

My personal portfolio website built with HTML, CSS, and Vanilla JavaScript.

## Sections
- **Hero** — intro with CTA buttons
- **Skills** — full tech stack
- **Projects** — all 9 featured projects with links
- **Experience** — work & education timeline
- **Contact** — links and email

## Features
- Smooth scroll navigation with active link highlighting
- CSS animations and hover effects
- Glassmorphism and gradient design
- Fully responsive — mobile-first

## Getting Started
Just open `index.html` in your browser, or:
```bash
npx serve .
```
